# Notiffy

Notification system for Ultrakill. This aims to provide a better alternative to blocking popups, for example when issuing update notices.

- **For regular users**: You don't need to manually install this if using a mod manager like r2modman
- **For mod developers**: see the [Wiki](https://github.com/end-4/ultrakill-notiffy/wiki)
